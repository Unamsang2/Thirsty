Why is Machine Learning imperative for us? One of the Purposes of this Project Prototype
Robots have replaced human beings in every area. Most of the tasks they have done were physical ones. However, as Machine Learning technology develops, computers began to deal with some of the complicated tasks considered to be too complex for them to do. Therefore, Machine Learning is not a topic of a science fiction. It is unavoidable, so now is the time to put my effort, time, attention, and everything for this tech.
Making a prototype which can amass and analyze legal documents by itself is one of the goals of this project. I have not decided what algorithm I am going to utilize to run Machine Learning. However, it is certain that Python would be the main programming language. This web-site is the first step of this project.
Put keywords or a sentence which you want to find relevant legal documents.
